# ICS-Doubletten-Scan
Thunderbird Addon für Lightning

ICS-Doubletten-Scan soll ein Tool zum Scannen des Dateiformats ICS sein. Es soll 2 Dateien nehmen und vergleichen können und die Differenzen anzeigen können.
